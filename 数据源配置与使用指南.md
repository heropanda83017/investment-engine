## 1. baostock — 主力K线+财务

### 配置
无需配置，直接 import。

### 使用
```python
import baostock as bs
lg = bs.login()
rs = bs.query_history_k_data_plus(
    "sh.600519", "date,close,volume,amount,peTTM,pbMRQ",
    start_date="2026-01-01", end_date="2026-05-27", frequency="d")
rows = []
while rs.next():
    rows.append(rs.get_row_data())
bs.logout()
```

### 代码位置
- `scripts/baostock_source.py`
- `scripts/data_optimizer.py` (OptimizedDataLayer 首选路由)

## 4. wudao MCP — 实时盘面/涨跌停/板块

### 配置
- MCP URL: `https://stock.quicktiny.cn/api/mcp-stream`
- Token: `lb_643b662d34b1...7fc8e7`
- 配置位置: `~/.hermes/profiles/ai-investor/config.yaml`

### 使用
```python
from strategies.wudao_mcp_source import (
    market_overview, limit_stats, hot_sectors,
    limit_up_ladder, capital_flow, get_all_market_data
)

# 市场概况（涨跌家数/温度）
ov = market_overview()

# 涨跌停统计
ls = limit_stats()

# 热点板块
hs = hot_sectors()

# 涨停梯队
ladder = limit_up_ladder()

# 资金流向
flow = capital_flow(flow_type="market")

# 一键获取全部
all_data = get_all_market_data()
```

### Pipeline 集成
S06 双模式: 直连优先 → 缓存兜底。缓存由 cronjob Step 0 生成。

### 注意
免费服务有 rate limit，每日首次调用间隔需 2 秒以上。

## 6. FactorHub — 208因子参考数据

### 配置
- API Key: `fh_8313ab79e2d4e52057f58ed0fb1e5748510417ac5fd0f71a`
- 配置位置: `config/config.json`

### 使用
```python
from strategies.factorhub_source import FactorHubClient
fc = FactorHubClient()

# 获取208因子参考数据（定义/分类/历史表现）
ref = fc.fetch_factor_values()
if ref:
    factors = ref.get("data", [])
    for f in factors[:5]:
        print(f['code'], f['name'], f['category'], f.get('sharpe_ratio'))
```

### 注意
`/factors/values` 端点不存在，仅支持 `/factors` 获取因子参考列表。
